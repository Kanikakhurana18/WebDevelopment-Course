Different types of CSS Selectors

  1. Element Selector:Any Element can be selected by its name
  2. Class Selector:Any class can be selected using(.)
  3. ID Selector:Any ID can be selected using(#) but id cannot start with number
  4. Child Selector:Direct child of any attribute can be selected using first write attribute then > sign(eg. di>p)
  5. Descendant Selector: Anywhere selector in that attribute works on it. Used by placing space between them
  6. Universal Selector: Selects all elements in file using(*) button
  7. Pseudo Selector: For links like active, visited,hovering links and its first child(eg.p:first-child)
